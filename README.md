# DiceOverlord
it's a nice, small and easy to understand IRC bot made for node.js


#Installation 
1.- download the source from https://github.com/luisoliv/DiceOverlord/ and extract it

2.- in Linux run the command "sudo apt install npm", in windows it's comes built-in with nodeJs 

3.- run the command "npm install irc" (this will install a necesary library dependency for the bot)

3.- then run it by "nodejs diceBot.js" , it will print some text about it's creator and you could see it would be running
